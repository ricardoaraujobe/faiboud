Usage: cvdKuhn2008 <fileName> <deficiencyType (0=protanopia | 1=deuteranopia | 2=tritanopia)> <exagerate-contrast (0=false | 1=true)> <max_number_quantized_colors>

Example:  cvdKuhn2008 natural_flowers.bmp 1 0 128

Default setup: fileName=natural_flowers.bmp, deficiencyType=1, exagerate-contrast=0, max_number_quantized_colors=128

DISCLAIMER: This distribution of the code works with .BMP files.
